# FixIt – Home Repair Service App

FixIt is a **Flutter mobile application prototype** designed to connect homeowners with verified repair professionals for quick and reliable home maintenance services. Users can explore service categories such as **plumbing, electrical, carpentry, and appliance fixing**, and navigate the booking process through an **intuitive interface**.

## Getting Started

### Prerequisites

* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* Android Studio / VS Code
* Device or emulator for testing

### Installation

1. Clone this repository:

```bash
git clone https://github.com/<username>/codesprint_app.git
```

2. Navigate to the project folder:

```bash
cd codesprint_app
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
